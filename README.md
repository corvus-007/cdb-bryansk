# Планы этажей

Убедиться, что в начале svg-файла присутствует строка:

`<?xml version="1.0" encoding="utf-8"?>`

Иначе WP не сможет загрузить файл.

## Область здания

* Fill (заливка) - `#dad9df`
* Stroke (обводка) - `#a7a495`

## Заливка помещения

* По умолчани - `#7e8ba2`
* Товары - `#505bda`;
* Услуги - `#a05f96`;
* Свободное помещение - `не_задана`
